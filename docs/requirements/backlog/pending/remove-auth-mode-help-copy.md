# Remove auth mode help copy

## 状態

未対応

## 優先度

P3

## 目的

ログイン / ユーザー登録画面の「利用したい操作を選んでください」という補助文言を削除し、認証画面の表示をより簡潔にする。
ログイン / 登録の切り替え自体はボタン表示で十分に伝わるため、重複した説明を減らす。

## 対象

- `frontend/src/features/auth/components/AuthScreen.vue`
- 日本語 / English の翻訳辞書
- 認証画面の design docs / UI specs
- 認証画面スクリーンショット
- 関連 unit / E2E の文言期待値

## 対応内容

- ログイン / 登録切り替え領域から「利用したい操作を選んでください」に相当する補助文言を削除する。
- 英語表示にも対応する補助文言がある場合は同様に削除する。
- 切り替えボタンの視認性、現在選択中の状態、余白のバランスが崩れないように調整する。
- 文言削除に合わせて design docs、UI specs、スクリーンショット、テスト期待値の更新要否を確認する。

## 完了条件

- ログイン画面とユーザー登録画面に冗長な操作説明が表示されない。
- ログイン / 登録の切り替え操作は引き続き分かりやすい。
- 日本語 / English の表示で未使用翻訳キーが残らない。
- 認証画面の関連テストとスクリーンショットが現行 UI と一致する。

## 根拠

ログイン / 登録の切り替えはセグメントコントロールとボタンラベルで意味が伝わるため、「利用したい操作を選んでください」は画面上の情報量を増やす割に新しい説明を追加していない。
