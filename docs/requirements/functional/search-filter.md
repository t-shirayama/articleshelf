# 検索・フィルタの機能要件

- 記事一覧を表示できる
- 検索とタグフィルタが使える
- タイトル / URL / メモ / タグ名を対象に検索できる
- 検索欄は記事一覧上部で広めに表示し、並び替えセレクトはややコンパクトにする
- 未読 / 既読 / すべて / お気に入りで絞り込める
- おすすめ度の複数選択で絞り込める
- 登録日と既読日の期間指定で絞り込める
- 適用中のフィルタ条件を一覧上部で確認できる
- 記事が増えた場合でも主要導線を維持して確認できる

具体的な一覧レイアウト、スクロール領域、スクロールバー表示の仕様は [UI仕様](../../specs/ui/README.md) と [コンポーネント設計](../../designs/components/README.md) を正本とします。
