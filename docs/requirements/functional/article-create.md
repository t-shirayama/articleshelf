# 記事追加の機能要件

- 記事を追加できる
- 入力項目は URL / タイトル / タグ / メモ / 既読日 / おすすめ度 / あとで読む とする
- URL は必須入力とし、未入力では保存しない
- URL から記事タイトル、概要、サムネイルを補完できる
- アクセスできない URL やエラー応答になる URL は保存しない
- 過去に追加したタグを選択でき、新しいタグも直接入力できる
- 記事追加時は、あとで読む状態または既読状態を指定できる
- おすすめ度を 0 から 5 の範囲で登録できる
- 重複 URL の場合は、登録済みの記事へ移動できる導線を表示する

具体的なフォーム挙動、タグ入力後のクリア、星表示、モーダル配置、エラー表示は [記事機能仕様](../../specs/features/articles.md)、[UI仕様](../../specs/ui/README.md)、[コンポーネント設計](../../designs/components/articles.md) を正本とする。
