# 記事詳細の機能要件

- 記事詳細で内容を確認・編集できる
- 記事詳細で URL / タイトル / 概要 / タグ / メモ / 既読日 / お気に入り / おすすめ度を編集できる
- 閲覧モードでメモが未入力の場合、メモがないことを表示する
- 記事詳細でも選択済みタグの削除、既存タグからの追加、新規タグ入力に対応する
- 記事を削除できる
- 記事削除時は確認ダイアログを表示し、ユーザーが確定した場合のみ削除する

具体的なタグ入力後のクリア、閲覧 / 編集切り替え時の幅、詳細画面の配置、未保存差分警告は [記事機能仕様](../../specs/features/articles.md)、[UI仕様](../../specs/ui/README.md)、[コンポーネント設計](../../designs/components/articles.md) を正本とする。
