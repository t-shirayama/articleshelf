# タグの機能要件

- タグを追加 / 選択 / 削除できる
- 記事に複数のタグを付与できる
- 選択済みタグ、既存タグからの追加、新規タグ入力を見分けて操作できる
- トップページ上部のフィルタ導線からタグで記事を絞り込める
- タグフィルタは複数選択に対応する
- 独立したタグ管理画面で、タグの追加、名称変更、統合、未使用タグの削除ができる
- タグ管理画面から、タグに紐づく記事一覧へ移動できる

具体的なタグ入力後のクリア、タグ選択 UI、タグ管理画面の表示と操作配置は [記事機能仕様](../../specs/features/articles.md)、[タグと検索仕様](../../specs/features/tags-and-search.md)、[UI仕様](../../specs/ui/README.md)、[コンポーネント設計](../../designs/components/articles.md) を正本とする。
