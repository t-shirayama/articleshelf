# タグ・検索の機能要件

## タグ

- タグを追加 / 選択 / 削除できる
- 記事に複数のタグを付与できる
- 選択済みタグ、既存タグからの追加、新規タグ入力を見分けて操作できる
- 既存タグからタグを追加した後は、既存タグ選択欄の値と検索文字列をクリアする
- トップページ上部のフィルタ導線からタグで記事を絞り込める
- タグフィルタは複数選択に対応する
- 独立したタグ管理画面は将来拡張として扱う

## 検索・フィルタ

- 記事一覧を表示できる
- 検索とタグフィルタが使える
- タイトル / URL / メモ / タグ名を対象に検索できる
- 検索欄は記事一覧上部で広めに表示し、並び替えセレクトはややコンパクトにする
- 未読 / 既読 / すべて / お気に入りで絞り込める
- おすすめ度の複数選択で絞り込める
- 登録日と既読日の期間指定で絞り込める
- 適用中のフィルタ条件を一覧上部で確認できる
- 記事が増えた場合でも、記事一覧領域内でスクロールして確認できる
- スクロールバーの有無で記事カード幅が変わらないようにする
