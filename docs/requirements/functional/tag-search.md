# タグ・検索の機能要件

## タグ

- タグを追加 / 選択 / 削除できる
- 記事に複数のタグを付与できる
- 選択済みタグ、既存タグからの追加、新規タグ入力を見分けて操作できる
- トップページ上部のフィルタ導線からタグで記事を絞り込める
- タグフィルタは複数選択に対応する

## 検索・フィルタ

- 記事一覧を表示できる
- 検索とタグフィルタが使える
- タイトル / URL / メモを対象に検索できる
- 未読 / 既読 / すべて / お気に入りで絞り込める
- おすすめ度の複数選択で絞り込める
- 登録日と既読日の期間指定で絞り込める
- 適用中のフィルタ条件を一覧上部で確認できる
- 記事が増えた場合でも、記事一覧領域内でスクロールして確認できる
