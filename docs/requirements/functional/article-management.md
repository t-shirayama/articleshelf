# 記事管理の機能要件

## 記事追加

- 記事を追加できる
- 入力項目は URL / タイトル / タグ / メモ / 既読日 とする
- URL は必須入力とし、未入力では保存しない
- アクセスできない URL やエラー応答になる URL は保存しない
- タイトルは任意入力とし、未入力の場合は URL から取得した記事タイトルを自動設定する
- URL から OGP を取得し、記事タイトルや概要を自動補完する
- 記事追加時のタグは、過去に追加したタグから選択できる
- 記事追加時のタグは、新しいタグを直接入力できる
- 記事追加時は「あとで読む」を選択でき、デフォルトでは未読として保存する
- 記事追加時におすすめ度を 0 から 5 の星で登録できる

## 記事状態

- 未読 / 既読を切り替えられる
- 既読日を記録できる
- お気に入り状態を管理できる
- おすすめ度を 0 から 5 の星で管理できる
- 記事一覧からもお気に入り追加 / 解除ができる
- 記事一覧を追加日 / 更新日 / 既読日 / タイトル / おすすめ度で並び替えられる

## カレンダー

- 記事を追加した日をカレンダーで確認できる
- 記事を既読にした日をカレンダーで確認できる
- 月ごとの追加数、既読数、積読差分を確認できる
- カレンダー上の記事から記事詳細へ移動できる

## 記事詳細

- 記事詳細で内容を確認・編集できる
- 記事詳細で URL / タイトル / 概要 / タグ / メモ / 既読日 / お気に入り / おすすめ度を編集できる
- 記事詳細でも選択済みタグの削除、既存タグからの追加、新規タグ入力に対応する
- 記事一覧上で未読から既読へ切り替えられる
- 記事一覧上で既読から未読へ戻せる
- 記事一覧上でステータスを切り替えた直後は元に戻せる
- 記事を削除できる
- 記事削除時は確認ダイアログを表示し、ユーザーが確定した場合のみ削除する
