# 機能要件

ArticleShelf の具体的な機能要件を、操作領域ごとに分割して管理します。

## 目次

- [記事追加](article-create.md): URL入力、OGP補完、重複URL、追加フォーム
- [記事状態](article-status.md): 未読 / 既読、お気に入り、おすすめ度、並び替え
- [記事詳細](article-detail.md): 詳細確認、編集、削除、詳細内タグ操作
- [Chrome 拡張機能](browser-extension.md): 現在ページを popup から直接保存するブラウザ拡張
- [カレンダー](calendar.md): 追加日 / 既読日の月次確認
- [タグ](tags.md): タグ追加 / 選択 / 名称変更 / 統合 / 未使用削除
- [検索・フィルタ](search-filter.md): 検索、タグフィルタ、状態 / 日付 / おすすめ度フィルタ
- [PDF インポート品質](pdf-import.md): PDF 抽出失敗パターン、補正フロー、代表確認ケース
