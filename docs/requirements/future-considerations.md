# 追加検討項目

将来的に追加したい機能や、まだ仕様を固めきっていない構想は、このファイルに蓄積します。
実装時期や仕様が具体化した段階で、必要に応じて `docs/specification/` 配下へ昇格させます。

## 将来的な拡張候補

- トップページからのステータス切り替え強化
  - 記事一覧上で未読から読了へ切り替えられる
  - 記事一覧上で読了から未読へ戻せる
- タグ追加 / 削除 UI・UX の改善
  - タグ操作を見た目で理解しやすくする
  - タグ追加と削除を迷わず行える操作導線にする
- カレンダー画面
  - 記事を追加した日を表示する表示モード
  - 記事を読了した日を表示する表示モード
- メモの Markdown 対応
  - 編集時は Markdown 記法で入力できる
  - 閲覧時は整形済み表示で確認できる
- Chrome 拡張機能によるクイック登録
  - 本番環境で閲覧中のページからすぐに未読登録できる
  - 本番環境で閲覧中のページからすぐに読了登録できる
- 画像やスクリーンショットの保存と参照
- OGP自動取得
- AI要約やハイライト
- ユーザー認証 / マルチユーザー対応
- クラウド同期
