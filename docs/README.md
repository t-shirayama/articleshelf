# ReadStack Docs

ReadStack の設計、要件、仕様、テスト、運用状況を整理する入口です。
`docs/` 直下にはこの README だけを置き、詳細文書は目的ごとのディレクトリに格納します。

## 目次

- [要件](requirements/README.md): プロダクト要件、機能要件、非機能要件、将来検討
- [仕様](specification/README.md): 機能仕様、API、データモデル、UI、認証、技術スタック、非機能仕様
- [アーキテクチャ](architecture/README.md): 全体構成、frontend / backend 詳細、DB、API フロー、実行環境
- [デザイン](designs/README.md): UI 方針、レイアウト、コンポーネント、画面別ルール、現行 UI スクリーンショット
- [テスト](testing/README.md): UT / IT / E2E の方針、ケース、CI/CD 連携
- [ステータス](status/project-status.md): 実装済み機能、差分、技術的注意点
- [タスクバックログ](status/task-backlog.md): 優先度つきの残タスク
- [デプロイ構成](deployment/README.md): 無料枠を中心にした公開・CI/CD 構成
