# 用語集

ArticleShelf の仕様、設計、UI で使う主要な用語を定義します。

| 用語 | 定義 |
| ---- | ---- |
| 記事 | ArticleShelf に保存する技術記事の単位。URL、タイトル、概要、サムネイル、タグ、メモ、状態などを持つ。 |
| OGP | Open Graph Protocol の略。URL からタイトル、概要、サムネイルなどの記事情報を補完するために参照するメタデータ。 |
| 未読 | まだ読んでいない、またはあとで読む対象として残している記事の状態。 |
| 既読 | 読み終えた記事の状態。既読日により振り返りの対象になる。 |
| 既読日 | 記事を既読にした日付。カレンダーや日付フィルタで振り返るために使う。 |
| お気に入り | 重要な記事やあとから見返したい記事に付ける目印。 |
| おすすめ度 | 記事の有用度や優先度を段階的に表す評価値。 |
| タグ | 記事を分類・検索するためのラベル。複数の記事に付与できる。 |
| メモ | 記事ごとに残す学び、要約、補足情報。Markdown 入力を前提に扱う。 |
| Backlog | 今後のタスク、残作業、構想段階のアイデア、技術的負債を集約する記録先。 |
