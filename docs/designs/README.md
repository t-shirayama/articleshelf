# ReadStack Design Assets

このディレクトリには、ReadStack の画面デザインイメージを格納しています。

## 含まれるデザイン

- `desktop_article_list.png`: デスクトップ版 - 記事一覧画面
- `desktop_article_detail_light.png`: デスクトップ版 - 記事詳細画面
- `add_article_modal.png`: 記事追加モーダル
- `mobile_article_list.png`: モバイル版 - 記事一覧画面

## 画面構成

### 記事一覧（desktop_article_list.png）

- 左サイドバー：フィルターナビゲーション（すべて、未読、既読、お気に入り）
- 中央メイン：記事のカード一覧表示
  - 記事サムネイル
  - タイトル、サイト名、既読日
  - タグ
  - 既読ステータスバッジ
- 右上：記事追加ボタン

### 記事詳細（desktop_article_detail_light.png）

- 背景：ダークグラデーション
- ヘッダー：戻る、タイトル、お気に入り、編集ボタン
- 記事情報エリア
  - タイトル
  - URL
  - タグ
- 記事イメージ表示
- 概要セクション
- 既読日・追加日の詳細パネル
- メモ入力欄

### 記事追加モーダル（add_article_modal.png）

- URL 入力欄
- タイトル入力欄
- タグ選択・追加フィールド
- メモ入力欄（任意）
- キャンセル・保存ボタン

### モバイル版（mobile_article_list.png）

- ヘッダー：アプリ名、検索バー
- 記事一覧を縦スクロール
  - 記事サムネイル（小さめ）
  - タイトル、サイト名
  - ステータスバッジ
- フローティングアクションボタン：記事追加

## 設計のポイント

- **レスポンシブデザイン**: デスクトップとモバイルで使いやすいUI
- **ダークモード対応**: 詳細画面はダークテーマを採用
- **視覚階層**: タグやステータスを色で区別
- **効率的なナビゲーション**: 検索とフィルターで素早くアクセス
- **一体型メモ**: 記事とメモを同画面で管理
