# 機能仕様

## 記事一覧

- すべての記事を一覧表示
- タイトル、ドメイン、タグ、読了ステータス、登録日、読了日を確認
- タイトル、ドメイン、タグ、読了ステータス、登録日、読了日、おすすめ度を確認
- 検索バーでタイトル・URL・メモを横断検索
- 左サイドバーで未読 / 読了 / すべて / お気に入りを選択してフィルタリング
- 上部のフィルタ導線からタグの複数選択、おすすめ度の複数選択、登録日範囲、読了日範囲で絞り込める
- 一覧画面のタイトルは左サイドバーや上部フィルタで選択中の条件に合わせて切り替わる
- 追加日 / 更新日 / 読了日 / タイトル / おすすめ度で並び替え
- 各記事カードからお気に入り追加 / 解除ができる
- 各記事カードから未読 / 読了を切り替えできる
- 未読から読了へ切り替えると、読了日は操作日になる
- 読了から未読へ戻すと、読了日は未設定になる
- ステータス切り替え直後はスナックバーから元に戻せる
- 記事が多い場合は記事一覧領域内でスクロールする
- サイドバー左下に学習継続を促す画像付きメッセージを表示する
- 学習継続メッセージは画面遷移や記事追加のタイミングでローテーション表示する

## 記事詳細

- 記事のURL、タイトル、概要、タグ、メモ、読了日、登録日時、更新日時、おすすめ度を表示
- メモを入力・編集できる
- 読了ステータスを切り替えられる
- 選択済みタグの削除、既存タグからの追加、新規タグ入力でタグを編集できる
- 読了日を更新できる
- おすすめ度を更新できる

## 記事追加

- URL、タイトル、タグ、メモ、読了日、おすすめ度を入力して追加
- URL は必須項目とし、未入力では保存しない
- アクセスできない URL やエラー応答になる URL は保存しない
- URLからOGPを取得してタイトル・概要・サムネイルを自動補完
- 既存記事の重複チェック
- タグは選択済みタグの削除、過去に追加したタグからの追加、新しい値の直接入力に対応する
- ステータス選択は表示せず、「あとで読む」チェックボックスで未読保存を指定する
- 「あとで読む」はデフォルトでチェック済みとする
- おすすめ度は 0 から 5 の星で登録する
- 画像添付は将来的な拡張として検討

## タグ管理

- タグを複数選択できる
- 記事にタグを付与可能
- よく使うタグを優先表示
- タグが多い場合はタグ領域内でスクロールする

## 検索・フィルター

- テキスト検索: タイトル、URL、メモ
- ステータスフィルタ: 左サイドバーで未読 / 読了 / すべてを選択
- タグフィルタ: 上部フィルタモーダルで複数タグを選択
- おすすめ度フィルタ: 上部フィルタモーダルで未設定を含む複数値を選択
- 登録日フィルタ: 上部フィルタモーダルで期間を指定
- 読了日フィルタ: 上部フィルタモーダルで期間を指定
- 並び替え: 追加日 / 更新日 / 読了日 / タイトル / おすすめ度
