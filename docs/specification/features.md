# 機能仕様

## 記事一覧

- すべての記事を一覧表示
- タイトル、ドメイン、概要またはメモ、タグ、既読ステータス、登録日、既読日、おすすめ度を確認
- タイトルは1行で省略し、概要またはメモは最大2行で表示する
- 概要やメモが未入力の場合、記事カードの概要欄は空欄で表示する
- OGPサムネイルを記事カード左側に表示し、取得できない場合はプレースホルダーを表示する
- 検索バーでタイトル・URL・メモ・タグ名を横断検索
- 左サイドバーで未読 / 既読 / すべて / お気に入りを選択してフィルタリング
- 上部のフィルタ導線からタグの複数選択、おすすめ度の複数選択、登録日範囲、既読日範囲で絞り込める
- 一覧画面のタイトルは左サイドバーや上部フィルタで選択中の条件に合わせて切り替わる
- 登録日 / 更新日 / 既読日 / タイトル / おすすめ度で並び替え
- 各記事カードからお気に入り追加 / 解除ができる
- 各記事カードから未読 / 既読を切り替えできる
- 未読状態の切り替えアイコンは灰色、既読状態は明るい青背景に白いチェックで表示する
- 未読から既読へ切り替えると、既読日は操作日になる
- 既読から未読へ戻すと、既読日は未設定になる
- ステータス切り替え直後はスナックバーから元に戻せる
- 記事が多い場合は記事一覧領域内でスクロールする
- スクロールバー表示の有無で記事カード幅が変わらないようにする
- サイドバー左下に学習継続を促す画像付きメッセージを表示する
- 学習継続メッセージは画面遷移や記事追加のタイミングでローテーション表示する

## 記事詳細

- 記事のURL、タイトル、概要、タグ、メモ、既読日、登録日時、更新日時、おすすめ度を表示
- 通常は閲覧モードで表示し、閲覧 / 編集トグルで編集モードへ切り替える
- 閲覧モードでメモが未入力の場合、メモ欄は空欄で表示する
- メモを入力・編集できる
- 既読ステータスを切り替えられる
- 選択済みタグの削除、既存タグからの追加、新規タグ入力でタグを編集できる
- 既存タグからタグを追加した後は、既存タグ選択欄の値と検索文字列をクリアする
- 既読日を更新できる
- おすすめ度を更新できる
- 保存と削除は右上の文字なしアイコンボタンで操作する
- 編集モードでも詳細ページの幅は閲覧モードと同じに保つ
- 編集モードで未保存の差分がある状態で別画面へ移動しようとすると、破棄して移動するか編集を続けるかを確認する

## カレンダー

- 左サイドバーからカレンダー画面へ移動できる
- 表示月を前月 / 次月へ切り替えられる
- 追加日モードでは、記事の登録日ごとに記事を表示する
- 既読日モードでは、記事の既読日ごとに記事を表示する
- カレンダーのタイトルと追加日 / 既読日のトグルは同じ行に配置する
- 月内の追加数、既読数、積読差分を表示する
- 土曜、日曜を色分けする
- 日付セルは6週分の固定グリッドとして表示する
- 日付セル内の記事をクリックすると記事詳細へ遷移する

## 記事追加

- URL、タイトル、タグ、メモ、既読日、おすすめ度を入力して追加
- URL は必須項目とし、未入力では保存しない
- URL 未入力のバリデーションエラー表示後でもキャンセルでモーダルを閉じられる
- アクセスできない URL やエラー応答になる URL は保存しない
- URLからOGPを取得してタイトル・概要・サムネイルを自動補完
- 既存記事の重複チェック
- 重複 URL の場合は登録済み記事詳細へ移動する導線を表示する
- タグは選択済みタグの削除、過去に追加したタグからの追加、新しい値の直接入力に対応する
- 既存タグからタグを追加した後は、既存タグ選択欄の値と検索文字列をクリアする
- ステータス選択は表示せず、「あとで読む」チェックボックスで未読保存を指定する
- 「あとで読む」はデフォルトでチェック済みとする
- おすすめ度は 0 から 5 の星で登録する
- おすすめ度は hover / focus 中の星まで色を変え、背景色は変えない
- 画像添付は将来的な拡張として検討

## タグ管理

- タグを複数選択できる
- 記事にタグを付与可能
- 既存タグはタグ一覧APIから取得し、追加モーダルと詳細編集で選択候補として使う
- 独立したタグ管理画面は未実装

## 検索・フィルター

- テキスト検索: タイトル、URL、メモ、タグ名
- ステータスフィルタ: 左サイドバーで未読 / 既読 / すべてを選択
- お気に入りフィルタ: 左サイドバーでお気に入りを選択
- タグフィルタ: 上部フィルタモーダルで複数タグを選択
- おすすめ度フィルタ: 上部フィルタモーダルで未設定を含む複数値を選択
- 登録日フィルタ: 上部フィルタモーダルで期間を指定
- 既読日フィルタ: 上部フィルタモーダルで期間を指定
- 並び替え: 登録日 / 更新日 / 既読日 / タイトル / おすすめ度
- タグ、おすすめ度、日付範囲の高度な絞り込みと並び替えはフロントエンド側で適用する
